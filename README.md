# TrackMoney - Сервис учёта и управления личными финансами

TrackMoney - это веб-приложение для учёта и управления личными финансами, созданное с использованием HTML, CSS, JavaScript и PHP с MySQL базой данных.

## Описание проекта

Проект представляет собой полнофункциональную систему учёта и управления личными финансами со следующими возможностями:

- Главная страница с обзором финансов
- Дашборд с аналитикой доходов и расходов
- Добавление и управление транзакциями
- Категоризация доходов и расходов
- Административная панель для управления категориями
- Система аутентификации пользователей
- Аналитика и отчеты

## Технологии

- **HTML5** - структура страниц
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript** - интерактивность и функциональность
- **PHP** - серверная логика и обработка данных
- **MySQL** - база данных
- **MAMP** - локальный веб-сервер для разработки

## Установка и запуск через MAMP

### Шаг 1: Установка MAMP

1. Скачайте MAMP с официального сайта: https://www.mamp.info/
2. Установите MAMP на ваш компьютер
3. Запустите MAMP

### Шаг 2: Настройка проекта

1. Скопируйте папку `TrackMoney` в директорию `htdocs` MAMP:

   - **Windows**: `C:\MAMP\htdocs\TrackMoney\`
   - **macOS**: `/Applications/MAMP/htdocs/TrackMoney/`
   - **Linux**: `/opt/lampp/htdocs/TrackMoney/`

2. Убедитесь, что структура папок выглядит следующим образом:
   ```
   htdocs/
   └── TrackMoney/
       ├── css/
       ├── db/
       ├── images/
       ├── includes/
       ├── js/
       ├── pages/
       ├── index.html
       └── README.md
   ```

### Шаг 3: Настройка базы данных

1. Откройте phpMyAdmin по адресу: `http://localhost:8888/phpMyAdmin/`
2. Войдите в систему:
   - **Имя пользователя:** `root`
   - **Пароль:** `root`
3. Создайте новую базу данных с именем `TrackMoney`
4. Импортируйте файл `db/db_kurs.sql` в созданную базу данных

### Шаг 4: Настройка подключения к базе данных

1. Откройте файл `includes/db_kurs.php`
2. Проверьте и при необходимости измените параметры подключения:
   ```php
   $host = 'localhost';
   $db   = 'kurs_project';
   $user = 'root';
   $pass = 'root';
   $charset = 'utf8mb4';
   ```

### Шаг 5: Запуск MAMP

1. Откройте MAMP
2. Нажмите кнопку **Start Servers**
3. Дождитесь, пока Apache и MySQL серверы запустятся (индикаторы станут зелеными)

### Шаг 6: Доступ к сайту

1. Откройте веб-браузер
2. Перейдите по адресу: `http://localhost:8888/TrackMoney/`
   - Если вы изменили порт в настройках MAMP, используйте соответствующий порт
   - По умолчанию MAMP использует порт 8888

### Шаг 7: Альтернативный способ доступа

Если стандартный порт не работает, попробуйте:

- `http://localhost/TrackMoney/` (порт 80)
- `http://127.0.0.1:8888/TrackMoney/`

## Возможные проблемы и решения

### Проблема: Страница не загружается

**Решение:**

- Убедитесь, что MAMP запущен
- Проверьте, что файлы находятся в папке `htdocs`
- Проверьте правильность URL

### Проблема: Ошибка подключения к базе данных

**Решение:**

- Проверьте, что MySQL сервер запущен в MAMP
- Убедитесь, что база данных создана и файл `db_kurs.sql` импортирован
- Проверьте настройки подключения в файле `includes/db_kurs.php`

### Проблема: Изображения не отображаются

**Решение:**

- Проверьте пути к изображениям в HTML файлах
- Убедитесь, что папка `images` находится в корне проекта

### Проблема: Стили не применяются

**Решение:**

- Проверьте пути к CSS файлам
- Убедитесь, что папка `css` находится в корне проекта
- Проверьте консоль браузера на наличие ошибок

### Проблема: JavaScript не работает

**Решение:**

- Проверьте пути к JS файлам
- Убедитесь, что папка `js` находится в корне проекта
- Проверьте консоль браузера на наличие ошибок

## Настройки MAMP (опционально)

### Изменение портов

1. Откройте MAMP
2. Перейдите в **Preferences** → **Ports**
3. Измените порты Apache и MySQL при необходимости
4. Перезапустите серверы

### Настройка виртуального хоста

1. Откройте файл `httpd.conf` в папке MAMP
2. Добавьте конфигурацию виртуального хоста
3. Перезапустите Apache
